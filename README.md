## 기능

1. 전체보기

   카드 전체를 보여줍니다.

   카드마다 우측 상단에 삭제할 수 있는 아이콘이 있습니다.

2. 완료/미완료 상태 선택

버튼을 누르면 상태가 변경됩니다.

3. 작성하기(+)

   클릭 시 작성하는 페이지로 이동합니다.
   항상 오른쪽 하단에 위치합니다.

4. 삭제하기

메인 페이지 or 상세 페이지에서 게시글을 삭제할 수 있습니다.

5. 수정하기

상세 페이지의 수정하기 버튼을 통해 게시글을 수정할 수 있습니다.

6. 댓글

댓글 Create, Read, Update, Delete가 가능합니다.
